| npm                                       | npx                                                 |
| ----------------------------------------- | --------------------------------------------------- |
| Installs packages                         | Executes packages                                   |
| Stores packages in `node_modules`         | Can run a package without permanently installing it |
| Used when you need the package repeatedly | Used for one-time execution                         |
| Package remains in your project           | Package may be downloaded temporarily and then used |


npm

npm stands for Node Package Manager.

Used to install dependencies.


npx

npx stands for Node Package Execute.

Used to run a package directly.
