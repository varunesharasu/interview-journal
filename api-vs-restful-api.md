An API (Application Programming Interface) is a general way for two software systems to communicate.

| API                               | RESTful API                                   |
| --------------------------------- | --------------------------------------------- |
| General concept                   | Specific architecture style                   |
| Can use different protocols       | Usually uses HTTP                             |
| May not follow any standard rules | Follows REST rules                            |
| Can return data in many formats   | Commonly uses JSON                            |
| May use functions or commands     | Uses HTTP methods like GET, POST, PUT, DELETE |


RESTful API:

GET /users → fetch users
POST /users → create user

# So, all RESTful APIs are APIs, but not all APIs are RESTful APIs.
