| GET                                  | POST                             |
| ------------------------------------ | -------------------------------- |
| Used to **retrieve data**            | Used to **send/create data**     |
| Data is sent in the URL              | Data is sent in the request body |
| Visible in browser URL               | Not visible in URL               |
| Limited data size                    | Can send large data              |
| More secure for large/sensitive data | Better for sensitive data        |
| Can be bookmarked                    | Cannot usually be bookmarked     |
