Python Data Types Syntax
List

Ordered and changeable collection.

my_list = [1, 2, 3, 4]
Tuple

Ordered but unchangeable collection.

my_tuple = (1, 2, 3)
Dictionary (dict)

Stores data in key-value pairs.

my_dict = {
    "name": "John",
    "age": 25
}
Set

Unordered collection of unique values.

my_set = {1, 2, 3, 4}
String

Text data type.

name = "Hello"
Integer

Whole numbers.

num = 10
Float

Decimal numbers.

price = 10.5
Boolean

True or False values.

is_active = True
None Type

Represents no value.

data = None
