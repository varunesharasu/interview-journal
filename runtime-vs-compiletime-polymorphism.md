| Compile-time Polymorphism                  | Runtime Polymorphism                  |
| ------------------------------------------ | ------------------------------------- |
| Achieved during compilation                | Achieved during program execution     |
| Also called **Method Overloading**         | Also called **Method Overriding**     |
| Same method name with different parameters | Same method in parent and child class |
| Faster                                     | Slightly slower                       |
| Decision made by compiler                  | Decision made at runtime              |



| Compile-time Error                | Runtime Error                                        |
| --------------------------------- | ---------------------------------------------------- |
| Occurs during compilation         | Occurs while program is running                      |
| Found by compiler                 | Found after execution starts                         |
| Prevents program from running     | Program compiles but crashes/errors during execution |
| Caused by syntax or type mistakes | Caused by invalid operations during execution        |
