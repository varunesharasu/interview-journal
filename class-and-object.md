what is class and object.
Class

A class is a blueprint or template for creating objects.

It defines:

Variables (data)
Methods (functions)

Example:

class Car {
    String color;
    
    void drive() {
        System.out.println("Car is running");
    }
}

Here, Car is a class.

Object

An object is a real instance of a class.

Example:

Car c1 = new Car();

Here, c1 is an object of the Car class.

Simple understanding:

Class = design/blueprint
Object = actual thing created from that blueprint

Example:

Class → Car
Object → BMW, Audi, Tesla cars created from that class definition.
