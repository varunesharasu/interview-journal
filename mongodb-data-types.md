what are the types of data can be stored in mongodb.

MongoDB can store these main types of data:

String — text values
Example: "name": "John"

Integer / Number — numeric values
Example: "age": 25

Boolean — true or false
Example: "isActive": true

Array — list of values
Example: "skills": ["Java", "Python"]

Object / Document — nested data
Example: "address": { "city": "Bangalore" }

Null — empty or unknown value
Example: "middleName": null

Date — date and time values

ObjectId — unique document identifier

Binary Data — files or binary content

Timestamp

Decimal

Regular Expression

MongoDB mainly stores data in BSON (Binary JSON) format.
